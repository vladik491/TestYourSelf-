# Проверь Себя (Check Yourself)

Казуальная викторина в стиле Pixel Art, разработанная для платформы Яндекс Игры. Проект объединяет фанатов поп-культуры, предлагая проверить знания в трех категориях: Фильмы, Сериалы и Аниме.

> **Статус проекта:** Полностью завершен технически (Gold Master). Не опубликован в стор только из-за вопросов авторского права на визуальный контент (используется как Portfolio Piece).

## Демонстрация
https://yandex.ru/games/app/489481?draft=true&lang=ru

## Особенности
*   **3 режима игры:**
    *   *Персонажи:* Угадай героя по изображению.
    *   *Связка:* Найди логическую связь между двумя элементами.
    *   *Общая проверка:* Классические вопросы на знание лора.
*   **Глубокая база контента:** Более 1600 вопросов и 1100 персонажей.
*   **Соревновательный элемент:** Глобальная таблица лидеров с защитой от накрутки (система убывающего вознаграждения).
*   **Атмосфера:** Уютный пиксель-арт (стилистика "Библиотека"), приятный саунд-дизайн и анимации интерфейса.

## Технический стек
*   **Движок:** Unity 6000.2.6f2
*   **Язык:** C#
*   **Платформа:** WebGL (Yandex Games SDK) / PC
*   **Архитектура:**
    *   Паттерн **State Machine** для управления состояниями игры (Меню -> Геймплей -> Результат).
    *   Адаптивная верстка UI для Web-билдов.

## Управление и Механики
*   **Жизни и Таймер:** Сложность варьируется (Легкий/Средний/Сложный). Ошибки отнимают жизни, таймер создает напряжение.
*   **Подсказки:** Система помощи (убрать неверный ответ) + монетизация через Reward Video (восстановление жизней/доп. подсказки).
*   **Управление:** Mouse-only (Full UI interaction).
